# 磨磨蹭蹭终于完成网站的建设

## 使用
**项目运行在9000端口，server启动在3000端口**

***先启动后端服务***

````
cd Blog-CMS/server
npm install
npm start
````

***启动服务后运行server/SQL/blog1.0.sql并更改server/db/index中的mysql相关配置***

***开发模式查看项目***
````
cd Blog-CMS
npm install
npm run dev
````



***打包上线***

````
cd Blog-CMS
npm run build
````
